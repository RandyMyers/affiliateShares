Models for Advertiser Features
Advertiser Model

Fields:
name: String, required - Name of the advertiser.
email: String, required - Email address of the advertiser.
password: String, required - Encrypted password.
createdAt: Date, default - Current timestamp.
updatedAt: Date, default - Current timestamp.
Relations:
affiliatePrograms: Array of references to AffiliateProgram.
AffiliateProgram Model

Fields:
advertiser: ObjectId, ref Advertiser, required - The advertiser creating the program.
name: String, required - Name of the affiliate program.
description: String, required - Description of the program.
commissionStructure: ObjectId, ref CommissionStructure, required - Commission structure for the program.
terms: String, required - Terms and conditions for affiliates.
createdAt: Date, default - Current timestamp.
updatedAt: Date, default - Current timestamp.
Relations:
creatives: Array of references to Creative.
CommissionStructure Model

Fields:
type: String, enum (flat, percentage, tiered), required - Type of commission.
rate: Number, required - Flat rate or percentage commission.
tieredRates: Array of objects:
tier: Number, required - Tier level.
rate: Number, required - Rate for the tier.
Relations:
affiliatePrograms: Array of references to AffiliateProgram.
Creative Model

Fields:
advertiser: ObjectId, ref Advertiser, required - The advertiser providing the creative.
type: String, enum (banner, link, datafeed), required - Type of creative.
url: String, required - URL of the creative (banner URL, download link, or datafeed URL).
description: String - Description of the creative.
createdAt: Date, default - Current timestamp.
updatedAt: Date, default - Current timestamp.
Relations:
affiliatePrograms: Array of references to AffiliateProgram.
